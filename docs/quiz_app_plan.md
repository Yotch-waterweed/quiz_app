# クイズアプリ開発方針

## 1. アプリの目的
`quiz.json` のデータを利用して、ユーザーがクイズを手軽に楽しめるシンプルなWebアプリケーションを作成する。

## 2. ターゲットユーザー
`quiz.json` に記載されているテーマ（OSI & TCP/IP モデル）に関心のある学習者や学生。

## 3. 主な機能
- `quiz.json` ファイルを動的に読み込み、クイズを表示する。
- ユーザーが選択肢をクリックすると、正誤判定を行う。
- 次の問題へ進む機能。
- 全ての問題が終了したら、最終的なスコア（正解数）を表示する。
- リセットして最初からやり直す機能。

## 4. 使用する技術
- **フロントエンドのみ**で完結させる。サーバーサイドの実装は不要。
- **HTML:** アプリケーションの基本的な骨格を定義する。
- **CSS:** シンプルで分かりやすいデザインを適用する。
- **JavaScript (Vanilla JS):** 
    - `fetch` API を使用して `quiz.json` を非同期で読み込む。
    - クイズのロジック（問題の表示、正誤判定、スコア計算など）を実装する。
    - DOM操作によって画面の表示を更新する。

## 5. デザインの方向性
- **ミニマル & クリーン:** 学習の妨げにならない、シンプルで集中しやすいデザインを目指す。
- **レスポンシブ:** スマートフォンでもPCでも快適に利用できるよう、画面サイズに応じてレイアウトが最適化されるようにする。

## 6. 開発ステップ案
1. **HTML/CSS:** 基本的なレイアウト（問題文、選択肢、結果表示エリア）を作成する。
2. **JavaScript:** `quiz.json` を読み込み、コンソールに表示する処理を実装する。
3. **JavaScript:** 1問目の問題と選択肢を画面に表示する処理を実装する。
4. **JavaScript:** 選択肢がクリックされた際の正誤判定ロジックを実装する。
5. **JavaScript:** 次の問題を表示する処理と、最終的な結果を表示するロジックを実装する。
