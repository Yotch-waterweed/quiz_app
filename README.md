# クイズアプリ

`quiz.json` のデータを読み込んで表示する、シンプルなWebクイズアプリケーションです。

## 概要

このアプリは、OSI参照モデルとTCP/IPモデルに関するクイズを提供します。
HTML, CSS, Vanilla JavaScriptのみで構築されています。

## 使い方（ローカルでの実行方法）

1.  このリポジトリをクローンまたはダウンロードします。
2.  ターミナルでこのディレクトリに移動します。
3.  以下のコマンドでローカルWebサーバーを起動します。
    ```bash
    python3 -m http.server
    ```
4.  ブラウザで `http://localhost:8000` にアクセスします。

## クイズデータ

クイズの問題と答えは `quiz.json` ファイルで管理されています。このファイルを編集することで、クイズの内容を自由に変更できます。
